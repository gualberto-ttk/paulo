Projeto: Sabor Rápido

Como executar:

1. Abra a pasta no VS Code
2. Abra o terminal
3. Instale o Flask:
   pip install flask
4. Execute:
   python app.py
5. Abra no navegador:
   http://127.0.0.1:5000
